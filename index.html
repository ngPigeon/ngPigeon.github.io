ngPigeon

A web application framework for web developers who want to build a simple web app easily in a short time.

Why use ngPigeon?

Easy to use

Generate essential web application components by using a single custom HTML element tag with SQL statement.

Fast Web App Development

Developers not need to write code to implement components. All you need is just download ngPigeon modules and attach to your web application and use it.

Powerful Web Application Components

ngPigeon provides powerful web application components to give better experiences to end users.

ngPigeon Modules

Pigeon Table

Pigeon Chart

Generate interactive charts to represent raw data graphically. Ease end users to understanding on large data set more easily.
